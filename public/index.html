    Introduction

    Welcome to the ToDoAPI documentation! ToDoAPI is a powerful and easy-to-use RESTful API designed to streamline the management of your tasks and to-do lists. This backend service provides essential CRUD (Create, Read, Update, and Delete) operations, including GET, POST, DELETE, and PATCH actions, to handle your to-do items effortlessly. Each to-do item is composed of a title, description, and an isDone property, allowing you to keep track of your tasks and maintain organization in your daily life. With ToDoAPI, developers can now seamlessly integrate comprehensive task management functionalities into their applications, ensuring an efficient and organized user experience.

    This endpoint retrieves all ToDos.

    HTTP Request

    GET https://todo-api-h8ov.onrender.com/api

    Response Body

    Field Type Description
    message string A message describing the result
    data array A Array object containing information of all ToDos

    Create ToDo

    Response

    {
  "message": "To-do created successfully",
  "toDo": {
    "id": 4,
    "title": "Clean the house",
    "description": "Vacuum, mop the floors, and dust the furniture",
    "isDone": false
  }
}

    This endpoint is used to create a new ToDo

    HTTP Request

    POST https://todo-api-h8ov.onrender.com/api

    Request Body

    Parameter Required Type Description
    title Required string The title for the ToDo
    description Required string The description for the ToDo
    isDone will be false by default, no need to send this key in the request body.

    Response Body

    Field Type Description
    message string A message describing the result
    toDo object A JSON object containing information about the new ToDo

    This endpoint deletes a specific ToDo.

    HTTP Request

    DELETE https://todo-api-h8ov.onrender.com/api/{id}

    URL Parameters

    Parameter Type Description
    {id} number The id of the ToDo

    Response Body

    Field Type Description
    message string A message describing the result
    toDo object A JSON object containing the deleted information about the ToDo

    This endpoint updates a specific ToDo.

    HTTP Request

    PATCH https://todo-api-h8ov.onrender.com/api/{id}

    URL Parameters

    Parameter Type Description
    {id} number The id of the ToDo

    Request Body

    Parameter Required Type Description
    title Optional string The title for the ToDo
    description Optional string The description for the ToDo
    isDone Optional boolean A boolean value that indicates whether the task is complete or not
    At least one field should be sent to the update request in the request body.

    Response Body

    Field Type Description
    message string A message describing the result
    toDo object A JSON object containing the information about the ToDo
